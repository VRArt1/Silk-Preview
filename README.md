# Silk Preview
A vibe coded Cocoon FE theme previewer.

A work in progress theme previewer for CocoonFE. Please excuse the lackluster read me segment while I get this set up!

Instructions:
Download repository and run Silk Preview.py.

| Key | Control |
| --- | --- |
| Tab | Frame Cycle |
| Shift Tab | Reverse Frame Cycle |
| L | Load Theme |
| R | Refresh |
| U | UI Toggle |
| S | Screenshot |
| Ctrl S | Screenshot Device Only |
| + | Zoom In |
| - | Zoom Out |

Bonus instruction: You can right click on the default folder to change its color!
